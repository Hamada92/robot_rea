# Robot

Robot game simulation gem.

## Installation

```ruby
gem install 'robot_rea'
```

## Usage

Run the executable:
```
robot
```

Try some commands: 

```
PLACE 0,0,NORTH
MOVE
REPORT
LEFT
REPORT
```

To configure the table size, update the file `config.yaml`

To run the specs, clone the repo then run `bundle exec rspec`
